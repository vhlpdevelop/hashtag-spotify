# 🎵 Projeto Clone do Spotify 🎧  

Este é um projeto desenvolvido com **React**, **JavaScript**, **HTML** e **CSS**. O objetivo é criar uma interface que imita o Spotify, demonstrando conceitos básicos de **React**, como:  

✅ Uso de **componentes** para organização do código  
✅ Aplicação do padrão **PascalCase** na nomenclatura  
✅ Consumo de API utilizando **Axios.js**  

## 🚀 Tecnologias Utilizadas  

- [React](https://react.dev/)  
- [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)  
- [Axios.js](https://axios-http.com/)  
- [HTML](https://developer.mozilla.org/pt-BR/docs/Web/HTML)  
- [CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS)
